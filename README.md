# Capstone-Project-Module-2
Analisis operasional Transjakarta untuk mengatasi inefisiensi, mengoptimalkan distribusi layanan, dan memahami demografi pelanggan. Analisa ini mengevaluasi dampak pembayaran digital, waktu puncak, dan analisa pendapatan untuk meningkatkan pelayanan yang dapat ditindaklanjuti demi kenyamanan lebih dan memajukan transportasi umum di Jakarta
